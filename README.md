# F1 Тотализатор 2026 — Telegram Bot

Бот для приёма прогнозов на гонки Формулы 1 сезона 2026.

## Быстрый старт

```bash
# 1. Создай .env файл
cp .env.example .env
# Вставь токен бота и свой Telegram ID

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Запусти бота
python bot.py
```

## Функциональность (v1 — базовая)

| Команда / кнопка | Что делает |
|---|---|
| `/start` | Регистрация + главное меню |
| `📅 Календарь` | Все 22 гонки сезона, отметка спринтов |
| `🏁 Прогноз` | Выбрать гонку и сделать прогноз на P1/P2/P3 |
| `📋 Мои прогнозы` | Список своих прогнозов с набранными очками |
| `🏆 Лидерборд` | Общая таблица участников |
| `🏎 Гонщики` | Список всех 20 гонщиков сезона |
| `/result MON race VER LEC NOR` | (Admin) Внести результат гонки |

## Система начисления очков

Настраивается в `data/scoring.py`:

| Событие | Очки (по умолчанию) |
|---|---|
| Точное попадание в P1 | +25 |
| Точное попадание в P2 | +18 |
| Точное попадание в P3 | +15 |
| Гонщик на подиуме, но не на той позиции | +5 |
| Гонщик не попал на подиум | -3 |
| Бонус за точный порядок всего подиума | +10 |
| Бонус за всех трёх гонщиков (любой порядок) | +5 |
| Спринт | × 0.5 |

## Правила

- Прогноз принимается до **5 минут** до старта гонки/спринта
- За 1 час до старта бот пришлёт напоминание
- За 5 минут до старта бот уведомит о закрытии приёма

## Структура проекта

```
bot.py              — точка входа
config.py           — конфигурация
database.py         — SQLite (aiosqlite)
scheduler.py        — напоминания
data/
  drivers.py        — список гонщиков 2026
  calendar_2026.py  — календарь с датами
  scoring.py        — логика начисления очков
handlers/
  start.py          — /start, главное меню
  predictions.py    — прогнозы (ConversationHandler)
  leaderboard.py    — лидерборд + обработка результатов
  calendar_handler.py — календарь, список гонщиков
  admin.py          — admin /result команда
```
